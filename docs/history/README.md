# Cinema Paradiso Documentation History

This folder preserves the user-facing README/DOCS snapshots from tagged releases, including the early **10K Movie Library Organizer** era before the project became **Cinema Paradiso**.

These files are historical copies. They are kept for progress tracking and should not be treated as current installation or configuration instructions. For current usage, see the repository root `README.md`.
